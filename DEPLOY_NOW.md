# 🚀 立即部署 - 最简单方法

## 当前问题

SSH连接被拒绝，服务器要求密钥认证。

## ✅ 最快解决方案：使用腾讯云Web终端

### 步骤1：登录腾讯云控制台

1. 访问：https://console.cloud.tencent.com/
2. 登录您的账号

### 步骤2：打开Web终端

1. 进入 **轻量应用服务器** → **服务器**
2. 找到您的服务器（IP: 115.159.92.235）
3. 点击服务器名称进入详情页
4. 点击 **登录** 按钮（或 **远程登录**）
5. 使用Web终端直接操作

### 步骤3：在Web终端中运行部署命令

连接成功后，您会看到命令行提示符，然后运行：

```bash
# 下载并运行一键部署脚本
curl -o setup-lighthouse.sh https://raw.githubusercontent.com/muller530/in-nutri-site/main/scripts/setup-lighthouse.sh
bash setup-lighthouse.sh
```

脚本会自动完成所有部署工作！

## 或者：配置SSH密钥（可选）

如果您想使用SSH命令行连接：

### 1. 生成SSH密钥

```bash
ssh-keygen -t rsa -b 4096 -C "your-email@example.com"
# 按回车使用默认路径
```

### 2. 查看公钥

```bash
cat ~/.ssh/id_rsa.pub
```

### 3. 在腾讯云控制台添加密钥

1. 进入 **轻量应用服务器** → **密钥**
2. 点击 **创建密钥** → **导入已有公钥**
3. 粘贴公钥内容
4. 保存后，绑定到您的服务器

### 4. 使用密钥登录

```bash
ssh root@115.159.92.235
```

## 推荐：直接使用Web终端

**最简单、最快的方法就是使用腾讯云Web终端**，无需配置SSH密钥，直接登录后运行部署脚本即可。

## 部署脚本会做什么？

一键部署脚本会自动：
- ✅ 安装 Node.js 18
- ✅ 安装 Git、PM2、Nginx
- ✅ 配置防火墙
- ✅ 克隆项目代码
- ✅ 安装依赖
- ✅ 配置环境变量（会询问域名）
- ✅ 构建项目
- ✅ 初始化数据库
- ✅ 启动应用
- ✅ 配置 Nginx

整个过程大约需要5-10分钟。
